    Slice

    Represents a slice of the subTokens of some Group. This is just a view of it, so taking e.g. tail is O(1). Most parser functions act on a Slice and call other functions on sub-slices.

    Static Method Summary

    Static Public Methods

    group(group: *): *: Group

    Slice representing all subTokens of a Group.

    Member Summary

    Public Members
    public

    loc: Loc

    Slice keeps track of changes to loc as sub-slices are made, so most parser functions will call tokens.loc when constructing the MsAst.
